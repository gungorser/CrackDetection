Crack Detection User interface project

For adding new images, click 'Images' For adding new datasets, click 'Datasets'

Usage:

Firstly, user should add new mages to image gallery. After then, different sets of images, which is called as 'datasets' can be created.

After datasets are successfully created, images on these datasets can be sent to calculation service. Click 'Calculations', then select one or more algorithms to calculate.

Note:Algorithms are predefined. New algorithms will be developed soon.
